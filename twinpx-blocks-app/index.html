Блоки для сайтов от Twin px

Приложение «Блоки для сайтов от Twin px» предлагает уникальные и полезные элементы для вашего сайта.

Используйте его, чтобы добавить такие блоки, как всплывающая кнопка, бегущая строка, всплывающее окно пользовательского соглашения, облако тегов и блок с социальными аккаунтами, где вы можете настроить иконки и ссылки самостоятельно для каждого аккаунта.
